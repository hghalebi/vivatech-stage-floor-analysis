# VivaTech Publish Media Kit

This folder turns the validated claim bank into copy-ready publishing assets.

- `claim_evidence_map.csv`: every hook mapped to evidence, caveat, and card file.
- `x_thread.md`: thread draft.
- `linkedin_post.md`: longer post draft.
- `editorial_brief.md`: thesis, red lines, and recommended visual order.
- `executive_viral_brief.md`: boardroom-ready thesis, post hooks, and red lines.
- `publish_index.html`: browsable kit with cards and copy blocks.

Source counts: 3153 companies, 1133 speakers, 355 matched speakers.
