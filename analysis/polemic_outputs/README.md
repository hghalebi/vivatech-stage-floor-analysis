# VivaTech Polemic Outputs

This folder contains second-stage, social-ready VivaTech claims and visuals generated from the normalized company and speaker CSVs.

- `polemic_claims.csv`: ranked claim bank with evidence and caveats.
- `social_cards/`: 1080x1350 SVG cards for sharing or screenshotting.
- `hall_attention.csv`, `label_attention.csv`, `top_tag_pairs.csv`, `duplicate_company_names.csv`: supporting derived tables.
